macOS Other User Bypass

How to bypass NAP Lockdown browser using macOS user accounts and a handy tool

Note that if you have Filevault enabled, you will not be able to access any of your apps or files in the other user account, rendering this guide effectively useless
Also note that this guide has a *very* small chance of you getting stuck in your second user account if NAPDisabler crashes (which it shouldn't) and NAP disables the lock screen now that NAPDisabler is no longer protecting it. If this happens, force restart your computer.

Requirements

  • Latest lockdown browser
  • Admin user account
  • NAPDisabler Tool - Click to download

Steps

  1. Open the NAPDisabler.dmg file and drag NAPDisabler to Applications if you haven't already
  2. Go to your Applications folder in Finder, right click NAPDisabler and click Open
  3. If you are unable to open the app, you may need to go to Settings > Privacy & Security > Scroll Down > Open Anyways; Don't worry, the app is safe, I just didn't pay for a $150 code signing signature so Apple makes my life harder.
  4. Once the app is open, you don't have to do anything, just make sure Quickuser and Enablelockscreen are both enabled, then open Settings > Users & Groups > Add User and create a new user account
  5. Switch to the new user and complete the setup wizard, then sign out and switch back to your current account
  6. Open NAP Locked down browser on your computer, accept the prompts to close everything
  7. To switch to your other user, press the Touch ID button as you would (unsure for macs that don't have touch id sorry), hover over your profile picture and switch to your new user
  8. You can now freely open any apps you want! To switch back, press the touch id button again.
